# MVT (Mobile Verification Toolkit)

The Mobile Verification Toolkit (MVT) was developed by the Amnesty International Security Lab to facilitate the consensual forensic analysis of Android and iOS devices, for the purpose of identifying traces of compromise. Versions for iOS and Android are both available.

The documentation includes guidance for interpretation of results.
